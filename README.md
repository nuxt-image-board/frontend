# Web_v4
***REMOVED***のフロントエンドサイド(Nuxt.js)  
SSRなので、Productionで実行する場合は全ファイルを揃えて送る

## Build Setup

``` bash
# 必須ライブラリをインストールする
$ npm install

# Devlopment環境をlocalhost:3000で起動する
$ npm run dev

# Production環境で起動する
$ npm run build
$ npm run start
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
